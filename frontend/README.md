# ChatGPT_Clone

A full-stack ChatGPT clone built with React, Node.js, Express, and AI API integration, featuring real-time conversations, chat history management, and a modern responsive UI.

ChatGPT Clone is a full-stack AI-powered chat application that allows users to interact with an intelligent conversational assistant in real time. The project is built using modern web technologies and provides a clean, responsive user interface for seamless communication.

# Features

1.AI-powered conversational chatbot

2.Real-time messaging experience

3.Chat history management

4.Responsive and modern UI

5.REST API integration

6.Scalable full-stack architecture

# Tech Stack

Frontend: React.js, CSS, Bootstrap/Material UI

Backend: Node.js, Express.js

Database: MongoDB

AI Integration: Gemini API

Version Control: Git & GitHub

# Goal

The objective of this project is to learn and implement AI-powered conversational interfaces while gaining hands-on experience with frontend development, backend APIs, database management, and third-party AI service integration.
